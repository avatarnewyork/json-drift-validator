# Security Policy

## Supported Versions

Security updates will be applied to:

- The latest `v1` release
- The main branch

## Reporting a Vulnerability

If you discover a security vulnerability related to:

- JSON parsing
- Wildcard evaluation
- Arbitrary code execution
- Data leakage in logs
- Unexpected exposure of secrets

Please **do NOT open a public issue**.

Instead, contact:

**Avatar New York**  
Email: security@avatarnewyork.com

We will investigate promptly and work with you on a responsible disclosure process.

## Handling Sensitive Data

This action processes JSON inputs which may contain:

- Application configuration  
- API keys  
- Infrastructure data  

**Never include real secrets in bug reports or pull requests**.

Mask or redact sensitive fields before sharing.

Thank you for helping keep this project secure!
